# DraftMind
基于多模态大模型的工程数字图纸智能管理系统

大家接受邀请后，请按流程同步代码

1.克隆仓库 git clone https://github.com/LingeringDream/DraftMind.git

2.切分支开发 严禁直接在 `main` 提交！每人新建自己的分支，例如 `git checkout -b feat/算法-张三`

3.提交与合并 开发完成后 Push，并在 GitHub 创建 **Pull Request (PR)**，队长 Code Review 通过后合并至 `dev` 分支。

4.提交信息规范 `git commit -m "feat: 完成图纸切片预处理模块"` 或 `fix: 修复正则匹配bug`
